## 🚀 How to Use

### Quick Start
```bash
# Navigate to repository
cd /path/to/blackroad-os-codex-infinity

# Source any light system
source .trinity/greenlight/scripts/memory-greenlight-templates.sh
source .trinity/yellowlight/scripts/memory-yellowlight-templates.sh
source .trinity/redlight/scripts/memory-redlight-templates.sh

# View complete documentation
cat .trinity/README.md
```

### GreenLight Examples
```bash
# Announce new work
gl_announce "my-api" "REST API Development" "1) Design 2) Build 3) Test" "Backend services"

# Log a deployment
gl_deployed "api-service" "v1.0.0" "production" "Initial release"

# Coordinate with another Claude
gl_coordinate "claude-frontend" "claude-backend" "Need API endpoint documentation"
```

### RedLight Examples
```bash
# List available templates
ls .trinity/redlight/templates/

# Copy a template to start building
cp .trinity/redlight/templates/blackroad-ultimate.html ./my-landing-page.html

# Edit and deploy!
```

### YellowLight Examples
```bash
# Load infrastructure templates
source .trinity/yellowlight/scripts/memory-yellowlight-templates.sh

# Access BlackRoad Codex
source .trinity/yellowlight/scripts/trinity-codex-integration.sh
```
